# Introdução

Este projeto foi realizado como Tech Challenge da quarta fase do curso de Pós-Graduação em Software Architecture, ministrado pela FIAP. O objetivo dessa fase é aplicar os conceitos de Microsserviços, SAGA, comunicação entre serviços, bancos de dados isolados, testes com cobertura obrigatória e BDD.

Este repositório inteiro é dedicado à documentação do projeto, incluindo diagramas, decisões arquiteturais e instruções de uso.

A documentação está organizada em pastas e arquivos Markdown numerados para facilitar a navegação.

Além disso, cada arquivo contém links de navegação para os próximos conteúdos. Apenas clicando em "Próximo" no final de cada arquivo você irá navegar por toda a documentação em ordem lógica.
